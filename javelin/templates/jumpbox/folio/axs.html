Telco asset management software

The solution

AxS is a big data analytics solution that aims to provide operational insights into telecommunication network in a fast turnaround time. It provides highly automated and cross-departmental data analysis capabilities for the operators.

Its use-cases include helping operators to select areas for investment and resource allocation, pinpoint faults that affect revenue, analyze impact of marketing campaigns.

Our involvement

We work on all areas of the technical stack. The stack consists of a distributed data processing pipeline that collects, validates, cleans and aggregates telco network data from their mediation servers, an in-memory data grid that allows ultra-fast querying of terabyte quantity of processed data, a RESTful service that performs queries and returns results, a light-weight web frontend that displays key performance indicators and interactive visualizations.

We also handle the deployment of the software onto AWS and bare-metal hardware, the infrastructure provisioning and the configuration manangement.

AxS is powered by:

  • Javascript / Angular.js / D3.js / Twiter Bootstrap
  • Scala / akka actors
  • Gridgain
  • Python / SaltStack
